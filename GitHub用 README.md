# 探索アルゴリズムの実装 (Search Algorithms)

このリポジトリでは、Pythonを用いた基本的な探索アルゴリズム（線形探索・二分探索）の実装を行っています。

## 含まれているアルゴリズム

### 1. 線形探索 (Linear Search)
リストの先頭から末尾に向かって、順番に1つずつデータを確認していく最もシンプルな探索方法です。
- **イメージ**: 国語辞典の1ページ目から順番に言葉を探していくような方法。
- **特徴**: データが順不同でも機能しますが、データ量が増えると探すのに時間がかかります。
- **計算量**: $O(n)$

### 2. 二分探索 (Binary Search)
探索する範囲を半分ずつに絞り込みながらデータを探す、非常に高速な探索方法です。
- **イメージ**: 国語辞典を真ん中で開き、探している言葉が「あ〜さ行」か「た〜わ行」かを判断し、さらにその半分を開く……という作業を繰り返す方法。
- **特徴**: 非常に高速に検索できますが、**データがあらかじめソート（並び替え）されていること**が前提条件となります。
- **計算量**: $O(\log n)$

## 実行環境
