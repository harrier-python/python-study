# Git学習メモ（2026/05/20）

## 今日やったこと
- Git for Windowsをインストールした
- Git Bashの動作確認をした
- 基本コマンドを実行した

## 実行したコマンド
- `git config --global user.email "..."` ユーザー設定
- `git config --global user.name "..."` ユーザー名設定
- `touch disk_usage.py` 空ファイル作成
- `git add disk_usage.py` ステージング
- `git status` 状態確認
- `git commit -m "最初のコミット"` コミット実行

## 学んだこと
Gitは「ファイルのタイムマシン」。
作業ディレクトリ → ステージング → リポジトリ の3段階で管理する。
